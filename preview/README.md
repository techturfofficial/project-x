## 🎨 Theme 1: Mint Aurora

**Primary Colors:** `#0D9488` (Deep Teal) | `#E8F5E9` (Soft Mint)  
**Accent:** `#F3E8FF` (Soft Lavender)  
**Background:** Dark Teal Gradient (Luxury Dark)

### Design Rationale
A sophisticated dark theme featuring teal and mint tones, creating a premium, calming aesthetic perfect for tech-forward professionals. The gradient background provides visual depth while maintaining focus on content.

### Visual Characteristics
- **Cards:** Frosted glass effect with teal undertones
- **Text:** Cream primary text on dark background
- **Glow Effects:** Mint and teal radial gradients
- **Highlights:** Lavender accents for interactive elements
- **Hero Sphere:** Animated gradient mixing teal and mint

### Color Palette
- Background Base: `#051A17`
- Glass Panel: `rgba(13, 148, 136, 0.08)`
- Text Primary: `#FDFBF7`
- Text Secondary: `rgba(253, 251, 247, 0.6)`
- Accent: `#F3E8FF`

### Desktop Preview Description
Clean, modern interface with a centered layout featuring left navigation sidebar, main content area, and right activity panel. The hero section displays prominent typography with gradient text highlight. Three statistical cards are arranged horizontally at the bottom.

### Mobile Preview Description
Single-column layout with sidebar and panels hidden. Hero content takes full width, stats stack vertically. Bottom input bar remains accessible with touch-friendly button sizing.

### Accessibility Notes
- **Contrast Ratio:** 8.5:1 (WCAG AAA compliant)
- **Focus States:** Visible border outlines on interactive elements
- **Color Independence:** Information not solely conveyed by color
- **Font Sizes:** Minimum 16px on mobile for legibility
- **Touch Targets:** 48px minimum for interactive elements

## 🚀 Quick Start

1. Open any theme HTML file in your browser:
   - `theme-1-mint-aurora.html`
   - `theme-2-lavender-aurora.html`
   - `theme-3-coral-aurora.html`
   - `theme-4-mint-cream.html`
   - `theme-5-teal-luxe.html`

2. **Switch Themes:**
   - Click the colored buttons in the top-right
   - Press keys 1-5 on your keyboard
   - Use arrow keys to cycle through themes

3. **Responsive Testing:**
   - Resize browser to test breakpoints
   - Test on mobile devices (480px width)
   - Check tablet view (768px width)
